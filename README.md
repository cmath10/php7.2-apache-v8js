# Simple docker build for v8js

Build containers:

```bash
$ bash docker-cli.sh build
```

Run containers:

```bash
$ bash docker-cli.sh up
```

Exec terminal:
```bash
$ bash docker-cli.sh console
```

Stop containers & remove
```bash
$ bash docker-cli.sh down
```

Under container terminal to test v8js at work:

```bash
$ php v8.php
```
